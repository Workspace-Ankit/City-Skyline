# 🌆 City Skyline

A creative and responsive city skyline built entirely with HTML and CSS. This project was developed as part of the **Responsive Web Design Certification** by [freeCodeCamp](https://www.freecodecamp.org/), and demonstrates fundamental web design concepts such as Flexbox, positioning, and CSS variables.

---

## ✅ Features

- 🎨 Pure HTML & CSS — no JavaScript used
- 🧱 Flexbox-based layout for both background and foreground buildings
- 🏙️ Layered depth effect using positioning
- 🌈 Custom CSS variables for theme consistency
- 📱 Fully responsive design (fits all screen sizes)
